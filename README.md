<h1 align="center"> Ninja Mo </h1>

<p align="center">
  <strong>A 2D game about a Ninja that kills enemies to save the villagers.</strong>
</p>

---
## (Controls)

| (Key) | (Action) |
| :--- | :--- |
| W / UP Arrow |(Move Up) |
| S / Down Arrow | (Move Down) |
| A / Left Arrow | (Move Left) |
| D / Right Arrow | (Move Right) |
| Space | (Attak Enemies) |

## The used programs to make the Game:
* **Godot Engine 4**: To make the game
* **Audacity**: To edit the soundeffects
* **programming language**: Godot script
